Installation Debugging Solutions

This to be a list of solutions for installation issues with this project:

  1. Fix node-gyp issues: Follow the steps in https://github.com/nodejs/node-gyp#installation. You don't need to install node-gyp but make sure that you have the correct environment setup for your OS.

  2. Test installation against latest npm npm/node version

  3. Fix Caching Issues

    rm -rf node_modules &&
    npm install &&
    npm run electron-rebuild &&
    npm start

  4. If you're on Windows, try using cmder

Debugging Release Builds​

cross-env DEBUG_PROD=true npm run build
cross-env DEBUG_PROD=true npm start
